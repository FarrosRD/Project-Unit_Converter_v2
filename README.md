# Project-Unit_Converter_v2

Aplikasi Unit Converter berbasis Python dengan antarmuka GUI.  
Proyek ini dibuat untuk keperluan akademik (tugas pemrograman dasar).

## Fitur
- Jenis Unit Konversi:
  - Currency
  - Length
  - Speed
  - Mass
  - Area
  - Data
  - Temperature

## Persyaratan
- Python 3.x
- Library pada `requirements.txt`

## Cara Menjalankan
```bash
pip install -r requirements.txt
python main.py
